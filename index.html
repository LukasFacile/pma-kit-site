How It Works

1

Install & Setup

Run undertow setup. The wizard walks you through API keys and risk preferences.

2

Agents Discover

SAGE researches markets, finds patterns, and proposes strategies. CRITIC validates them.

3

Strategies run in paper mode first. Watch your simulated portfolio grow before risking real money.

Go Live

Upgrade to Pro to enable live trading. CONDUCTOR manages capital allocation and risk automatically.

$ undertow setup
$ undertow serve
# Dashboard at http://localhost:8000
