FAQ

Is this financial advice?

No. Undertow is software — a tool for prediction market research and trading. It is not a registered investment advisor. See the full Disclaimer.

Can I lose money?

Yes. Every prediction market contract settles at $0 or $1. The risk management system limits losses, but no system can guarantee profits. Only trade with money you can afford to lose.

How much does the LLM cost to run?

The system has a configurable daily budget (default $5/day). Most monitoring runs with zero LLM cost. Typical daily cost during development was $2-5 total across all agents.

Is Kalshi legal?

Kalshi is regulated by the U.S. CFTC and available to eligible U.S. residents. Check kalshi.com for your state.

Can I add my own strategies?

Yes. Drop a Python file in ~/.pma/strategies/ or publish a package. Any class extending Strategy is auto-discovered.

What happens when a strategy loses?

That is expected and built into the design. 5 consecutive losses triggers auto-demotion. 30% drawdown triggers a halt. During testing, the system caught and stopped a 14-trade losing streak automatically.

What makes this different from a trading bot?

Bots execute a fixed strategy. Undertow is a self-correcting research system. The agents argue with each other, reject bad strategies, and continuously discover new ones. The starters are seeds — Quant finds the rest.

How does the Claude integration work?

Undertow includes an MCP (Model Context Protocol) server with 10 tools. Add one line to your Claude Desktop or Claude Code config and Claude can access your markets, run backtests, check positions, and analyze strategies through natural conversation. You use your own Claude subscription — we don't resell AI access.

Why one-time instead of a subscription?

Undertow runs on your machine with zero server costs to us. Charging monthly for locally-run software would be dishonest. You pay once, you own it. The optional Edge add-on ($19/mo) is for curated strategy content and community — ongoing value for ongoing price.

What do I get after I buy?

A license key by email within minutes. Install with pip install undertow-ai, run undertow setup, activate your license, and start the agents.
